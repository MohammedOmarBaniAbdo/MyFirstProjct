# Rock Paper Scissors Game (C++)

A simple console-based Rock Paper Scissors game written in C++.

## Description
This project is a small game where the player competes against the computer in multiple rounds of Rock, Paper, Scissors.

The player chooses one of the following options:
1 - Stone (Rock)
2 - Paper
3 - Scissors

The computer randomly selects its move, and the program determines the winner of each round.

At the end of the game, the program displays:
- Number of ties
- Number of computer wins
- Number of player wins
- Final game result

The game also changes the console color depending on the result of each round.

## Features
- Random computer choices
- Multiple rounds gameplay
- Score tracking
- Console color changes based on result
- Option to replay the game

## Technologies Used
- C++
- Console Application
- Standard Library (iostream)

## How to Run

Compile the program:

g++ main.cpp -o game

Run the program:

./game
